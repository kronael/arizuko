v0.40.9 shipped. Two features + 4-round codebase refine:

**Engagement** (spec 5/G): `chat_reply_state.engaged_until + engaged_folder` — bot stays listening
in a `(jid, topic)` for 10m after each reply or mention. `resolveOrEngaged` gateway fallback
delivers inbounds without a fresh @mention. `engage`/`disengage` MCP tools with three-arm authz.

**Per-surface output styles** (spec 5/Y): runner derives `<platform>-<surface>` from JID shape;
ships `slack-{dm,channel,thread,pane}`, `telegram-{dm,group}`, `discord-{dm,channel}`, `web.md`.

renderer in emaid, extracted `sseHeaders` in webd, removed `pageTop`/`dashNav` from dashd,
dropped `signState` wrapper + vestigial bool param in auth, unified JID parser in chanlib.

Spec IDs: `I-modality-envelope` → `X`, `O-output-styles` → `Y` (base58 compliance).
