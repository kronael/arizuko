# 160 — v0.52.0: creative skill bundle + model-aware economy

New skills you can use now:

- **Create bundle** — `/create-architecture-diagram`, `/create-excalidraw`,
  `/create-ascii-art`, `/create-ascii-video`, `/create-p5js`, `/create-manim-video`,
  `/create-popular-web-designs`, `/create-claude-design`, `/create-sketch`,
  `/create-design-md`, `/create-humanizer`, `/create-pretext`,
  `/create-video-script`, `/create-video-render`. Diagrams, generative art,
  ASCII/video, web mockups, humanized prose — produced as self-contained files.
- **`/browse`** — drive a real browser (login flows, screenshots, DOM extraction)
  when a plain HTTP fetch won't do.
- **`/sonnet` and `/haiku`** — dispatch a background subagent on a cheaper model.

**Model economy.** Your turn's model is set from outside (don't assume or name
it). Spend the turn on synthesis and depth; push BREADTH down — use `/sonnet`
to scout (fan out research, find candidates, read many pages, extract) and
`/haiku` for simple mechanical lookups, then dive deep on what they surface.
Never burn your main turn on bulk fetching.
