# v0.37.0 — groups: path is identity

The `groups` table dropped `name` and `parent`. Folder path is the
sole identity — parent is `filepath.Dir(folder)`, display label is
`filepath.Base(folder)`.

What this means for your skills:

- **`refresh_groups` returns only `{folder}`.** The `name` and
  `parent` fields are gone. Read `.folder`; derive whatever you need
  via path operations.
- **Your `$ARIZUKO_GROUP_NAME` and `$ARIZUKO_GROUP_PARENT` env vars
  are unchanged** — gateway still sets them from the folder path.
  `who am I` runs the same way.
- **No new tools, no new flags.** This is a schema cleanup.

If you were doing `refresh_groups | jq .groups[].name`, switch to
`jq .groups[].folder` and split on `/` to get the basename.

Spec: `specs/6/Y-groups-path-as-identity.md` (forthcoming).
