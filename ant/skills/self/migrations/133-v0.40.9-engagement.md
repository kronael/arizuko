# v0.40.9 — engagement + per-surface output styles

Schema migration `0058-engagement-column.sql` adds two columns to
`chat_reply_state`: `engaged_until` (RFC3339Nano deadline, NULL = idle) and
`engaged_folder` (routing target for the engaged pair). Pre-existing rows stay
idle until the next bot reply or mention populates them.

New per-surface output style files ship in `ant/output-styles/`:
`slack-{dm,channel,thread,pane}.md`, `telegram-{dm,group}.md`,
`discord-{dm,channel}.md`, `web.md`. Selected automatically by the runner on
each spawn; no operator action required.
