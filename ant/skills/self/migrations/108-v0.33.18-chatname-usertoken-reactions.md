# v0.33.18 — Channel name context, Discord user token, reaction guidelines

Agent now receives the source channel name (`#general`, `"My Group"`) directly
in message context — no tool call needed to identify the channel.

Discord adapters support `DISCORD_USER_TOKEN` for user-account authentication.

Agent gained reaction guidelines: when to react vs reply, anti-patterns, and
mirroring the user's emoji register.
