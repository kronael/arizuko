# 162 — v0.52.0: reply threading, mention promotion, reliability

Key changes in this release:

- **Reply threading** — `reply` lands in a platform thread (Slack/Discord/Telegram);
  `send` stays top-level. Per-group `thread_replies` preference.
- **Thread @mention promotion** — replies inside a thread where the bot already
  posted are promoted to `verb=mention` automatically (spec 5/L).
- **Slack @mentions fixed** — `app_mention` events now delivered; outbound converts
  CommonMark → mrkdwn.
- **Turn retry** — runs killed by OOM/timeout reschedule with 10s backoff.
- **Observability** — OTLP spans + metrics. Set `OTEL_EXPORTER_OTLP_ENDPOINT` to export.
- **Instance model** — `ARIZUKO_DEFAULT_MODEL` in `.env` sets the agent model;
  per-group override now actually reaches the container.
- **Mid-turn progress** — `<status>` lines stream to channel; graceful timeout nudge.
