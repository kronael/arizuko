# v0.33.26 — dispatch on continuations, verify-before-send, tool discipline

Four agent-side fixes targeting the support-chat failure modes:

**`/resolve` dispatches on continuations too.** Previously continuation
turns skipped recall AND dispatch, so `/support` (and any other matched
skill) fired exactly once per case and never again. Single bug
explained: `/support` not sticky across multi-turn cases, facts/
write-only on continuations, source-first rule never re-enforced,
derivation-not-source-read after turn 1. Resolve now skips recall on
continuations (still cheap) but always re-dispatches.

**`/support` adds verify (phase 4) + investigate (phase 5).** Before
sending, re-grep the cited source for the claimed value. If verify
fails: don't send — go to Investigate. Re-verify entity ID literally,
re-verify source path against `~/facts/sources.md`, re-verify field
name, re-grep `~/facts/` for past corrections, re-run Gather from
scratch. If still no verification: send "I can't verify — checked X,
Y, Z; alternatives: A, B" — never a confident wrong answer.

**`/support` cite block moved to end of message** (per operator
instruction). Lead with answer, body in the middle, citation at the
bottom — reader scans answer first, finds source last when verifying.

**`/migrate` enumerates groups via `refresh_groups`** instead of the
broken `/workspace/data/groups/*/` glob. The old glob matched one
level only; nested subgroups like `atlas/support` were silently
skipped. Affected migrations may need to be re-run for affected
subgroups; the broadcast announce step was already correct.

**Tool discipline rule** in `ant/CLAUDE.md` — on 429 / timeout / empty
result, retry once with backoff before reporting unavailable.
Enumerate alternatives (inspect_*, facts/sources.md, refs/) before
declaring an API doesn't exist. "Not accessible" without enumeration
is a contract break.

**Skill format alignment**: skills touched here (resolve, support,
migrate, persona) drop the silently-ignored `when_to_use` frontmatter
field per canonical Anthropic spec. Description carries USE/NOT
pattern. The 43-skill full overhaul is queued separately.
