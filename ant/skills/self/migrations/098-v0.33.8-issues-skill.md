# 098 — issues skill

The "how to record user-reported bugs" procedure has moved out of
`CLAUDE.md` into a dedicated `/issues` skill at
`~/.claude/skills/issues/SKILL.md`.

Behavior unchanged: when a user reports a bug or complaint you can't
fix this turn, append a one-line bullet to `~/issues.md` (lowercase,
group root only). Operator periodically consolidates and wipes.

What changed: discoverability. `CLAUDE.md` no longer carries the
procedure body — it's a one-line pointer. Read the skill the first
time you're about to log; subsequent calls are cheap. No data
migration; no file moves; nothing in `~/issues.md` changes.
