# v0.34.0 — slakd (Slack adapter)

No skill changes. Release marker so auto-migrate fires and the v0.34.0
CHANGELOG blockquote is broadcast to all groups.

New adapter `slakd` ships as a deployable channel — bot-token,
single-workspace, HTTP Events API via proxyd at `/slack/events`. JIDs
take the `slack:<workspace>/<kind>/<id>` shape (kind ∈ `channel` / `dm`
/ `group`). Threads ride in `Topic = thread_ts`; reactions, file
attachments and `:thumbsdown:` dislike all round-trip.

Operator setup: `template/web/pub/howto/slack.html`. Spec:
`specs/2/l-slakd.md`.
