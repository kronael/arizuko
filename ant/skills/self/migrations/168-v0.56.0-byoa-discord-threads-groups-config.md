# v0.56.0 — BYOA secrets + Discord thread replies + groups config

BYOA: bring your own API keys at `/dash/me/secrets` — stored encrypted, injected at spawn.

Discord threads now stay live after the bot's first reply (no `@mention` required for follow-ups).

`thread_replies=false` now suppresses new-thread creation only; in-thread replies still land in the thread.

Per-group behavioral settings (`model`, `thread_replies`, `observe_window_*`, `open`) migrated to a single `config` JSON column.
