# v0.33.14 — Products, Discord mention-only, invite subgroups

## What changed

- `arizuko create --product <name>` seeds a new group from `ant/examples/<name>/`
- 8 product templates shipped: personal, support (atlas), trip, strategy, pm, reality, creator, socials
- `groups.product` DB column added
- Discord guild channels: @mentions trigger replies, ambient traffic stored as context only
- Invite links can now create a new subgroup at any depth (trailing-slash target_glob)
- `group add` CLI lowercases folder name by default

## Action required

None. Existing groups get `product = 'assistant'` by default.
