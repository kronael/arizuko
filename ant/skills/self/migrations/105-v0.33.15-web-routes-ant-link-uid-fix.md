# v0.33.15 — Web routes, ant link, UID fix, voice

## What changed

- New MCP tools: `set_web_route`, `del_web_route`, `list_web_routes` — control per-path access on your web host (`public`/`auth`/`deny`/`redirect`)
- Unknown paths now require login by default; use `set_web_route` to make paths public
- Slink renamed "ant link" in UI; mobile CSS improvements, SSE auto-reconnect
- `send_voice` MCP tool: synthesize and deliver voice on Telegram/WhatsApp/Discord
- Container workspace now writable after group creation (UID 1000 fix)
- Direct invite links land on the group's ant link instead of the onboarding dashboard

## Action required

None. Existing groups are unaffected.
