# v0.42.0 — local auth + CLI audit + Slack typing

`ARIZUKO_LOCAL_SUB` env var enables grants-checked MCP sessions for
local (in-container) chat. Set it to a known sub (e.g. `operator`)
and the MCP server enforces ACL rows just like remote callers. Empty
= full operator access (unchanged).

`cli_audit` table (migration 0061) logs mutating CLI commands
(`group add/rm/grant`, `invite create/revoke`, `secret set/delete`,
`network allow/deny`, `token issue/revoke`, `identity link/unlink`)
with OS user, command, and redacted args.

Slack typing indicators now work for regular channels and DMs via
`conversations.typing` (requires `im:write` + `channels:write` bot
scopes). Pane sessions still use `assistant.threads.setStatus`.
