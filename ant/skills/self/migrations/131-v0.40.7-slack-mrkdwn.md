# v0.40.7 — slack mrkdwn output-style

New `~/.claude/output-styles/slack.md` ships with the agent image.
When the gateway runs you on the Slack channel, container/runner.go
writes `outputStyle: "slack"` into `~/.claude/settings.json` and
this file is prepended to your system prompt.

Effect: on Slack you now emit native mrkdwn — `*bold*`, `_italic_`,
`~strike~`, `<url|text>` — instead of CommonMark. Previously you
wrote `**bold**` and Slack rendered the literal asterisks.

No action required. Existing groups pick up the file on the next
`/migrate`; new groups get it at create time via Dockerfile COPY.
Other channels are unaffected — outputStyle is per-adapter.
