# v0.40.2 — self-service WhatsApp re-pair + uniform reaction-mentions

Operator clicks a button instead of running docker by hand. And every adapter now treats reactions on the bot's messages the same way.

- whapd re-pair lives at `/dash/channels/whatsapp/pair` — operator-only (`**` super-grant). Code returned once, expires in 60s, rate-limited 5/hr. Spec 8/15.
- Reactions and text-replies pointing at the bot now get promoted to `verb=mention` in the gateway for ALL adapters (Discord, Telegram, WhatsApp, Slack), not just Discord. `verb=mention`-filtered routes finally work everywhere. Spec 6/J.
- slack-team product docs got a "same shape on Discord" section — most of the product maps 1:1 (channels, threads, per-user memory); 3 deltas listed (no sidebar pane, mention syntax, JID namespace).
- `@<unknown-folder>` from the previous release: the gateway passes it to the agent — you can `delegate_group` if it's clearly a typo for an existing child.
