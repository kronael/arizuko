# v0.42.0 — dashboard overhaul + security hardening

Five new dashd pages (invites, model selector, skill toggle, grants viewer,
workspace links) and six security fixes (list_acl, inject_message,
escalate_group, tier-2 web mount, egress wildcard, purge fatal).

Secrets now encrypted AES-256-GCM at rest; set `SECRETS_KEY` env var
(independent of `AUTH_SECRET`) to rotate the key without touching JWT signing.

If you see the agent using a different model than expected, check
`groups.model` — the per-group selector now overrides the instance default.
