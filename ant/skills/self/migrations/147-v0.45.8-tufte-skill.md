# 147 — v0.45.8: tufte data-viz skill

New stock skill `tufte` — Edward Tufte's principles applied to any
chart, dashboard, KPI tile, sparkline, or infographic the agent
produces. Adapted from `github.com/aref-vc/tufte-claude-skill` (MIT).

Triggers on "make a chart", "visualize", "improve this chart",
"design a dashboard", `/tufte`, plus the usual viz keywords. Default
output is a single self-contained HTML file (inline SVG, no external
deps), suitable for `send_file` attachment. React/Recharts preset
included for projects that need it.

No behavior change to existing surfaces. Existing skills, route
table, MCP tools, REST endpoints unchanged.

Reference: `~/.claude/skills/tufte/SKILL.md`. Files: principles.md,
chart-selection.md, kill-list.md, checklist.md, presets/, plus
before-after.html and cheatsheet.html for in-browser reference.
