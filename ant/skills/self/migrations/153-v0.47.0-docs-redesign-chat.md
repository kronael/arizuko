# 153 — v0.47.0: docs redesign, persistent chat, platform-reply fixes

Mostly platform fixes plus a docs/chat refresh. A few things behave better
for your tools; nothing you must change.

## What changed under you

- **Bluesky**: your default `reply` / `like` / `delete` to an inbound mention
  now resolves. Before, the inbound id was a bare rkey and those calls
  silently failed; inbound ids are now full `at://` URIs, and `send`/`post`
  return the created post's URI so you can edit/delete it later.
- **Reddit**: the `delete` verb works now (the capability was just unadvertised).
- **Email**: replayed history carries the trusted/untrusted signal.
- **Feed verbs**: `quote` / `repost` now record into your conversation, so you
  see your own feed posts in context and can act on them.
- **Docs**: the operator docs at `/pub/arizuko/` were rebuilt into two clean
  layouts (a navigable reference + a guided tour). If you link users to docs,
  the section URLs are unchanged; the prior docs live at `/pub/arizuko/legacy/`.

No action required.
