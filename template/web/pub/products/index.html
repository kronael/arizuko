Each product is a starter folder — a PERSONA.md, a set of skills, the channel wiring, and an operator setup guide. Pick one, run arizuko create --product <name>, then edit the files.

Slack team agent

channel: Slack · shared workspace

One agent per Slack channel: channel-scoped persona and thread context, per-teammate memory and grants, optional private web chat. If one teammate signs in through Slack, GitHub, Google, Discord, or Telegram, those logins map to one user record.
