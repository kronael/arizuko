products

Each product is a starter folder — a PERSONA.md, a set of skills, the channel wiring, and an operator setup guide. Pick one, run arizuko create --product <name>, then edit the files.

Under the hood a product is no new machinery: it’s the same six-primitive pipeline every arizuko agent runs, with the folder contents and routing fixed for one job. That’s why these are yours to read, diff, and change — a product you own, not a blob you rent.

Slack team agent

channel: Slack · shared workspace

One agent per Slack channel: channel-scoped persona and thread context, per-teammate memory and grants, optional private web chat. If one teammate signs in through Slack, GitHub, Google, Discord, or Telegram, those logins map to one user record.

Reality agent (Rhias)

channel: Telegram · single-user

Long-running personal agent. Holds named threads for ongoing situations across weeks; structured reflection on demand; monthly scheduled sweep over diary + threads to surface what changed.
